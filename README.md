## Highlights

- PHP (Hypertext Preprocessor) is a server-side scripting language used to create dynamic web pages.
- PHP is embedded in HTML and processed on the server before sending output to the browser.
- PHP is widely used in frameworks like **Laravel** for backend development.
- Tools like **XAMPP**, **MAMP**, or **Laragon** are used to create a local development environment (currently using XAMPP).
- PHP files are served through `localhost` via a tool like Apache or Nginx (currently using Apache).
